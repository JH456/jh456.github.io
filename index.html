About Me

Hello! Welcome to my website! I'm a Computer Science student studying at the Georgia Institute of Technology with a specialization in internet security. Born in New Jersey, I studied at Robbinsville Highschool and participated on the software subteam (which I led during the 2015 season) for our local Robotics team, FRC 2590, Nemesis. Around this time, I first developed an interest in programming for creating video games which I had a lifelong passion for. After experimenting with that, I discovered an even greater passion for computer security. I realized that making video games isn't what interested me specifically; making fun projects in general interested me, and the recurring conflict with each project I took on came back to security. How can this robot be made safe? How do we stop people from exploiting glitches in this game? How can I password protect this website? So essentially, I think security is fun because it makes it possible to do other fun things. I hope to, after earning my degree, establish a career in cyber security so that I can help make these fun things possible. In pursuit of this goal, I have learned a large number of different programming languages including C, C++, Java, JavaScript, LabView, Python, PHP, and markup languages such as HTML and CSS. A friend and I are currently in the process of learning Ruby on Rails so that we can learn more about web frameworks and how to implement them in projects. Projects

I try to keep myself working on personal software projects. Doing this helps me to build my skills, while working on something that I have a vested interest in. Whether I'm learning a new language or just trying to have some fun with friends, a project always makes coding much more fun! Here, I'll talk about a few projects that I'm especially proud of.

Blackjack

The first language I started with was java, back in my freshman year of highschool. However, after a few days of struggling to figure out how NetBeans is supposed to work, I switched to C++, which I for some reason found easier to pick up. I haven't had too much use for that language throughout highschool and college though, so I try to keep it fresh. One project I worked on to that end is a console based blackjack game! A classic beginner's program, blackjack covers many basic programming techniques. It also gave me an excuse to use cool ascii box drawing characters to make the cards! Feel free to check out the full source code on my github account (giant symbol on the left)!

3D Graphics

While on the topic of C++, another program I've been working on is a basic 3D graphics engine. I'm especially excited about this, though I see it as more of a longterm project. Hopefully, I can eventually turn this into my very own video game! My interest in this started during a lecture in my freshman Linear Algebra class. My professor gave a lecture on computer graphics and how linear algebra can be used to give 2D objects a 3D appearance. So, I decided to use OpenGL's 2D library to make a basic 3D environment! Right now it's pretty low level, and there are some bugs and leaks, but I plan to work them out in the future as I learn more about how to address them.

IRC Bots

During my junior year of highschool, my friend got me to start talking with him over IRC. He made this really awesome bot for our channel, and helped me learn how to make one too! We coded our bots in JavaScript, because NodeJS has many packages that making coding a breeze. Developing these bots was some of the best fun I ever had coding! We made it so that they could dance (print out /o/ and \o\ alot), store user data, and even send Yo's to people using the Yo API. I made mine so that it could send Yo's in a loop, so that I could launch Yo flood attacks on people (Not really too malicious, but super annoying). Although the API could only send one Yo per minute to combat this, to some degree, the staggering makes it much more annoying. I would tell kids in my classes to make a Yo account and send a Yo to my bot (he could only send Yo's to people that Yo him first, so kind of like an annoying vampire). Once they did, I would send them 800 Yo's. It was great. Ultimately I made two JS IRC bots, Pann-IRC and DL-IRC. The first one was my fun bot that I put random features into, and the latter I used to download youtube videos using youtube-dl. Both of these bots are on my GitHub. However, I have some favoritism toward DL-IRC because the code for that one is neater, and the config is updated to be more complex and awesome.
